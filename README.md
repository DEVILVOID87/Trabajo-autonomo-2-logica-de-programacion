## Objetivo del proyecto
Desarrollar un programa en Python que permita generar contraseñas seguras y,
a partir de su análisis, reflexionar sobre el impacto de la ciberseguridad
como una tecnología clave en la sociedad actual y futura.

---

## Descripción del proyecto
Este proyecto se basa en el desarrollo de un generador seguro de contraseñas.
Además de generar contraseñas aleatorias, el programa permite analizar su nivel
de seguridad y visualizar escenarios futuros relacionados con el uso de
tecnologías digitales y la protección de la información personal.

---

## Funcionalidades
- Generación de contraseñas seguras
- Validación de longitud mínima
- Uso de estructuras lógicas y repetitivas
- Análisis del nivel de seguridad
- Reflexión sobre impacto social y futuro
- Exportación de resultados
